# Release Notes: 2026-Feb-18 (Wed)

## (H) Database String Trimming

Added trailing whitespace trimming to all database engines for consistent string value handling across PostgreSQL, MySQL, and SQLite.

### PostgreSQL

- Added [`postgresql_trim_trailing_whitespace()`](/elements/001-hydrogen/hydrogen/src/database/postgresql/query.c) helper function
- Applied trimming in [`postgresql_execute_query()`](/elements/001-hydrogen/hydrogen/src/database/postgresql/query.c) and [`postgresql_execute_prepared()`](/elements/001-hydrogen/hydrogen/src/database/postgresql/query.c)

### MySQL

- Added [`mysql_trim_trailing_whitespace()`](/elements/001-hydrogen/hydrogen/src/database/mysql/query_helpers.c) helper function
- Applied trimming in [`mysql_build_json_from_result()`](/elements/001-hydrogen/hydrogen/src/database/mysql/query_helpers.c), [`mysql_process_query_result()`](/elements/001-hydrogen/hydrogen/src/database/mysql/query_helpers.c), and [`mysql_process_prepared_result()`](/elements/001-hydrogen/hydrogen/src/database/mysql/query_helpers.c)

### SQLite

- Added [`sqlite_trim_trailing_whitespace()`](/elements/001-hydrogen/hydrogen/src/database/sqlite/query.c) helper function
- Applied trimming in [`sqlite_exec_callback()`](/elements/001-hydrogen/hydrogen/src/database/sqlite/query.c) and [`sqlite_fetch_row_data()`](/elements/001-hydrogen/hydrogen/src/database/sqlite/query_helpers.c)

This change ensures string values like "TEXTDATA    " are returned as "TEXTDATA" across all database engines, providing consistent behavior particularly for YugabyteDB which uses PostgreSQL pathways.
