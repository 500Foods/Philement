# Acuranzo Database Diagrams

This directory contains database schema diagrams for the Acuranzo database design, generated for various database engines.

## Available Diagrams

- `acuranzo-db2-acuranzo-1024.svg` - DB2 database schema diagram
- `acuranzo-mysql-acuranzo-1024.svg` - MySQL database schema diagram
- `acuranzo-postgresql-app-1024.svg` - PostgreSQL database schema diagram
- `acuranzo-sqlite-1024.svg` - SQLite database schema diagram

## Generation

These diagrams are automatically generated using the diagram generation scripts in the test suite.

## Design

The Acuranzo database design is a general-purpose schema supporting various application needs. See the main Acuranzo documentation for details on the schema structure and migration files.
