Tours are exactly what they sound like!

Selecting a Tour will popup a little window that describes elements on the page. It typically has 10-20 steps that explain what the main parts of the user interface do, what each of the buttons do, how these functions generally work, and so on. You can move back and forth through the steps or exit the tour at any time. There is a button that shows you all the tours that you have access to that is part of any tour. Normally, the Tour button will show you what Tours are available that are related to the particular Manager that you\'re working in at the time.
