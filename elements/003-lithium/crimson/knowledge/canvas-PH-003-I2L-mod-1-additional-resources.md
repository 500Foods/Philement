This course is by no means the only source of information on the Lithium project.

The project itself is opensource and published on Github as part of the Philement project.

An AI model has been integrated directly into Lithium. It has been equipped with nearly everything we have about Lithium, including this course, portions of the GitHub repository, additional developer documentation, and more. It can discuss nearly every aspect of Lithium in surprising depth.

The developers are (hopefully!) still alive and kicking and would love to hear from you if you have any questions or issues that you can\'t otherwise find an answer to.
