While we live in an age of touch screens and voice-controlled AI assistants, people working with a desktop app day-to-day, week-after-week, doing data entry and other less glamorous tasks often do the same things repeatedly. To help with that, an assortment of Keyboard Shortcuts has been created and curated for each Manager.

The Keyboard Shortcuts Icon:

This appears in the top-right collection with the rest of the \'command and control\' icons like Zoom, Fullscreen, and Close. Clicking the icon displays a popup list of Keyboard Shortcuts specific to that Manager, along with the list of Global Shortcuts that are always available app-wide.
